# OFFONIKA — Система тестирования сотрудников

Стек: **Next.js + TypeScript + Tailwind CSS + shadcn/ui + SQLite + Prisma ORM**.

## Возможности
- Роли: администратор и сотрудник.
- Авторизация по логину/паролю.
- Демо-тесты по процессам OFFONIKA (1С, касса, приёмка, возвраты, закрытие смены).
- Автоматический расчёт результатов: процент, статус (Сдал/Не сдал), ошибки, дата.
- Хранение результатов в SQLite.
- Правило компании про запрет изменения документов задним числом включено в вопросы.

## Локальный запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Инициализируйте БД:
   ```bash
   npm run db:push
   ```
3. Заполните демо-данными:
   ```bash
   npm run db:seed
   ```
4. Запустите приложение:
   ```bash
   npm run dev
   ```
5. Откройте `http://localhost:3000`.

## Демо-аккаунты
- Администратор: `admin` / `admin123`
- Сотрудник: `seller` / `seller123`

## Логотип
Поместите логотип OFFONIKA в `public/logo.png`. Если файл есть, он будет показан на входе и в шапке.
